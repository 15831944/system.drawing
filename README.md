# system.drawing

Partial c++ implementation of .NET System.rawing classes.

Currently supported classes, structures and enumerations are:

- Bitmap
- Color
- GraphicsUnit
- Image
- Point
- PointF
- Rectangle
- RectangleF
- RotateFlipType
- Size
- SizeF
- BitmapData
- ColorPalette
- ImageFormat
- ImageLockMode
- PixelFormat
